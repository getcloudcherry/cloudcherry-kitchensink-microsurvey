SDK has 3 methods that can be used to trigger the survey, Replace token with the actual token number.

Show

To show the survey programatically use the show method as shown below

window.CCSDK(“show”, token);

Hide

To hide the survey programmatically use the hide method as shown below

window.CCSDK(“hide”, token);

Destroy

To destroy the survey programmatically use the destroy method as shown Below

window.CCSDK(“destroy”, token);
